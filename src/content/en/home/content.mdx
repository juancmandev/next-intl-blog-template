# Next.js app router i18n blog template

## Start your blog in multiple languages!

This is a template developed with:

- [Next.js app router](https://nextjs.org/)
- [next-intl](https://next-intl-docs.vercel.app/)
- [contentlayer](https://contentlayer.dev/)
- [mdx](https://mdxjs.com/docs/)
- [tailwincss](https://tailwindcss.com/)
- [shadcn/ui](https://ui.shadcn.com/)

Try changing the language using the button in the header.

The content of this page is a .mdx file.

You can fork this template and use it for a project that requires content creation in multiple languages.
