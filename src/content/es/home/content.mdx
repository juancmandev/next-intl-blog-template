# Next.js app router i18n blog template

## ¡Comienza tu blog en múltiples idiomas!

Este es un template desarrollado con:

- [Next.js app router](https://nextjs.org/)
- [next-intl](https://next-intl-docs.vercel.app/)
- [contentlayer](https://contentlayer.dev/)
- [mdx](https://mdxjs.com/docs/)
- [tailwincss](https://tailwindcss.com/)
- [shadcn/ui](https://ui.shadcn.com/)

Intenta cambiar el idioma usando el botón en el header.

El contenido de esta página es un archivo .mdx.

Puedes hacer un fork a este template y utilizarlo para algún proyecto que requiera de creación de contenido en múltiples idiomas.
